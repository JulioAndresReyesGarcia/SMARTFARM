# SMARTFARM
SmartFarm es una aplicación orientada al sector ganadero, diseñada para pequeños y medianos productores que actualmente gestionan la alimentación de su ganado, sin apoyo tecnológico ni cálculos nutricionales precisos. Esta situación genera problemas como mala nutrición animal, desperdicio de alimento, incremento de costos y baja productividad.

FUNCIONES
•	Registro y gestión de animales con datos detallados.
•	Cálculo automático de la ración de alimentación diaria.
•	Recomendación de dietas generadas por IA en función de las características del animal.
•	Registro de producción de leche o incremento de peso.
•	Módulo de análisis de costos de alimentación.
•	Generación de reportes y gráficos de desempeño.
•	Notificaciones y alertas relevantes para el manejo del ganado.
